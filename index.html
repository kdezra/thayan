Derivational morphology

Adjective → adverb = Suffix -o
Adjective → noun (the quality of being [adj]) = Suffix -ab
Adjective → verb (to make something [adj]) = Suffix -iː
Noun → adjective (having the quality of [noun]) = If starts with vowel: Prefix is-
Else: Prefix isi-
Noun → adjective relating to noun (e.g. economy → economic) = If starts with vowel: Prefix n-
Else: Prefix ne-
Noun to verb = If ends with vowel: Suffix -jj
Else: Suffix -awjj
Verb → adjective (result of doing [verb]) = Suffix -id
Tending to = If ends with vowel: Suffix -b
Else: Suffix -ub
Verb → noun (the act of [verb]) = If ends with vowel: Suffix -m
Else: Suffix -im
Verb → noun that verb produces (e.g. know → knowledge) = If ends with vowel: Suffix -r
Else: Suffix -ur
One who [verb]s (e.g. paint → painter) = Suffix -asij
Place of (e.g. wine → winery) = If ends with vowel: Suffix -ba
Else: Suffix -uːba
Diminutive = If ends with vowel: Suffix -ʃu
Else: Suffix -awʃu
Augmentative = If ends with vowel: Suffix -r
Else: Suffix -ur
Opposite = If ends with vowel: Suffix -m
Else: Suffix -iːm
